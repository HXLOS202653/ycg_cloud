# 网络云盘系统 - 后端项目结构

## 项目概述

本文档详细描述了网络云盘系统后端项目的完整结构，基于Go语言构建的高性能、可扩展的云存储服务。

## 技术栈

- **编程语言**: Go 1.24.6
- **Web框架**: Gin 1.10.0
- **ORM框架**: GORM 1.30.1
- **数据库**: MySQL 8.0+ (主数据库)
- **NoSQL**: MongoDB 6.0+ (文件元数据)
- **缓存**: Redis 7.0+ (缓存和会话)
- **消息队列**: Redis Pub/Sub + Stream 7.0+ (异步任务)
- **对象存储**: MinIO / AWS S3 (文件存储)
- **搜索引擎**: MySQL 8.0+ 全文索引 (全文搜索)
- **实时通信**: WebSocket (聊天和通知)
- **监控**: Prometheus + Grafana
- **日志**: Logrus + ELK Stack
- **容器化**: Docker + Docker Compose
- **API文档**: Swagger/OpenAPI 3.0

### MongoDB 6.0 LTS 版本兼容性

**scripts/database/ 脚本适配要求：**

#### 1. version-check.sh（版本兼容性检查脚本）
```bash
#!/bin/bash
# MongoDB 6.0 LTS 版本检查脚本

# 检查MongoDB版本
check_mongodb_version() {
    local mongo_version=$(mongosh --version | grep -o 'v[0-9]\+\.[0-9]\+' | head -1)
    local major_version=$(echo $mongo_version | cut -d'.' -f1 | sed 's/v//')
    local minor_version=$(echo $mongo_version | cut -d'.' -f2)
    
    if [[ $major_version -eq 6 && $minor_version -ge 0 ]]; then
        echo "✅ MongoDB版本检查通过: $mongo_version (兼容6.0 LTS)"
        return 0
    else
        echo "❌ MongoDB版本不兼容: $mongo_version (要求: 6.0+)"
        echo "请参考《技术选型》文档安装MongoDB 6.0 LTS版本"
        return 1
    fi
}
```

        --db=${MONGO_DB:-ycgcloud} \

        if (serverStatus.version.startsWith('6.0')) {

            // 创建6.0兼容的集合和索引
