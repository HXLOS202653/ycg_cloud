# 网络云盘系统 - 字段映射规范（完整版）

## 📋 概述

字段映射规范定义了前端、后端、数据库之间的数据字段转换和映射关系，确保数据在不同层级之间的传输准确、一致、高效。本规范涵盖了数据库字段到API响应、API请求到数据库存储、前端显示到后端处理等各个环节的映射关系。

**本文档基于最新数据库设计（v2.0）更新，包含所有模块的完整字段映射规范。**

## 🔄 映射原则

### 1. 基本映射原则
- **一致性原则**: 相同含义的字段在各层保持语义一致
- **转换规范**: 严格按照各层命名规范进行字段名转换
- **类型安全**: 确保字段类型在转换过程中的准确性
- **完整性原则**: 关键字段不得在映射过程中丢失
- **安全原则**: 敏感字段需要适当的过滤和脱敏处理

### 2. 命名转换规则
```
数据库 (snake_case) ↔ Go后端 (camelCase/PascalCase) ↔ JSON API (snake_case) ↔ 前端 (camelCase)

示例转换链：
user_id (DB) → UserID (Go Struct) → user_id (JSON) → userId (Frontend)
created_at (DB) → CreatedAt (Go Struct) → created_at (JSON) → createdAt (Frontend)
is_active (DB) → IsActive (Go Struct) → is_active (JSON) → isActive (Frontend)
```

// 计算存储使用率
func calculateStorageUsageRate(used, quota uint64) float64 {
    if quota == 0 {
        return 0
    }
    return float64(used) / float64(quota) * 100
}

// 格式化文件大小
func formatFileSize(size uint64) string {
    const unit = 1024
    if size < unit {
        return fmt.Sprintf("%d B", size)
    }
    div, exp := uint64(unit), 0
    for n := size / unit; n >= unit; n /= unit {
        div *= unit
        exp++
    }
    return fmt.Sprintf("%.1f %cB", float64(size)/float64(div), "KMGTPE"[exp])
}

// 解析文件类型
func parseFileTypes(types *string) []string {
    if types == nil || *types == "" {
        return []string{}

    return strings.Split(*types, ",")
}

func parseFileTags(tags *string) []string {
    if tags == nil || *tags == "" {

func formatTime(t *time.Time) *string {
    if t == nil {

// 验证文件类型
func validateFileTypes(fl validator.FieldLevel) bool {
    types := fl.Field().String()
    if types == "" {
        return true
    }
    
    allowedTypes := []string{"jpg", "jpeg", "png", "gif", "pdf", "doc", "docx", "xls", "xlsx", "ppt", "pptx", "txt", "zip", "rar"}
    fileTypes := strings.Split(types, ",")
    
    for _, fileType := range fileTypes {
        fileType = strings.TrimSpace(fileType)
        if !contains(allowedTypes, fileType) {
            return false
        }
    }
    return true
}

// 验证存储配额
func validateStorageQuota(fl validator.FieldLevel) bool {
    quota := fl.Field().Uint()
    return quota >= 1024*1024 && quota <= 1024*1024*1024*1024 // 1MB - 1TB
}

## 📋 最佳实践

### 12.1 字段映射最佳实践

1. **命名一致性**
   - 数据库使用 `snake_case`
   - Go 结构体使用 `PascalCase`
   - JSON API 使用 `snake_case`
   - 前端 TypeScript 使用 `camelCase`

2. **类型安全**
   - 使用强类型枚举而非字符串常量
   - 指针类型用于可选字段
   - 时间统一使用 `time.Time` 和 RFC3339 格式

3. **安全考虑**
   - 敏感字段（如密码哈希）不在 API 响应中暴露
   - 使用 `omitempty` 标签避免空值传输
   - 实现字段级权限控制

4. **性能优化**
   - 使用懒加载避免 N+1 查询
   - 合理使用数据库索引
   - 实现分页和过滤
