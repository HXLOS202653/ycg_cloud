# 网络云盘系统 - 技术栈与兼容性分析

## 技术架构（经过兼容性验证）
- **前端**: 
  * **核心框架**: React 18.3.1 + TypeScript 5.6.3 + Ant Design 5.22.2
  * **状态管理**: Redux Toolkit 2.3.0 + TanStack Query 5.59.16 (原React Query)
  * **路由**: React Router 6.28.0
  * **构建工具**: Vite 6.0.1 + SWC 1.9.2
  * **HTTP客户端**: Axios 1.7.7
  * **实时通信**: Socket.io-client 4.8.1（支持WebSocket断线重连、消息压缩）
  * **文件处理**: react-dropzone 14.3.5（支持文件夹上传、拖拽排序）
  * **音视频**: WebRTC API（支持P2P通信、SFU架构）+ Web Audio API（波形可视化、降噪处理）
  * **存储**: IndexedDB（断点续传状态）+ localStorage（用户设置）
  * **UI组件**: 自定义文件预览器、进度条、波形显示组件

- **后端**: 
  * **核心框架**: Go 1.24.6 + Gin 1.10.0
  * **配置管理**: Viper 1.20.1（支持多格式配置文件、环境变量覆盖）
  * **WebSocket**: gorilla/websocket 1.5.3（支持消息压缩、心跳检测）
  * **ORM**: GORM 1.30.1（支持事务、批量操作、软删除、泛型优化）
  * **数据库驱动**: 
    - MySQL: gorm.io/driver/mysql 1.6.0
    - MongoDB: go.mongodb.org/mongo-driver 1.17.4
    - Redis: github.com/redis/go-redis/v9 9.12.0
  * **认证**: JWT（access token + refresh token机制）+ bcrypt（密码加密）
  * **任务队列**: 自定义队列系统（优先级、重试机制、死信队列）
  * **文件处理**: 分片上传、流式ZIP打包、文件类型检测

- **数据库**: 
  * **关系型**: MySQL 8.0.31（InnoDB引擎、全文索引、事务支持）
  * **文档型**: MongoDB 4.4+（消息存储、日志记录、大文件元数据，驱动版本1.17.4）
  * **缓存**: Redis 7.0+（分布式锁、会话存储、热点数据缓存、发布订阅、消息队列，客户端v9.12.0）

- **存储与CDN**: 
  * **对象存储**: 阿里云OSS SDK（支持分片上传、图片处理、CDN加速）
  * **本地存储**: 临时文件系统（分片缓存、合并处理）
  * **CDN**: 多节点智能路由、边缘计算支持

- **安全与监控**: 
  * **监控**: Prometheus 2.55.1 + Grafana 11.3.1（系统指标、业务指标可视化）
  * **日志**: 结构化日志 + 文件轮转（按大小/时间切割）
  * **安全扫描**: ClamAV（病毒检测）+ 自定义敏感内容过滤

- **部署与运维**: 
  * **容器化**: Docker 27.3.1 + Docker Compose 2.29.7（多服务编排）
  * **反向代理**: Nginx 1.27.2（负载均衡、SSL终结、静态资源缓存）
  * **SSL证书**: Let's Encrypt + Certbot（自动续期）
